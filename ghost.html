less mixin for ghost button with less Hat

link says boo Visit HTML5 Ninja

Syntax

Default value:

  • @color1:#3f6fa0
  • @color2:#fff
  • @height:36px
  • @fontSize:1em
  • @radius:5px
  • @fontFamily:"Arial"
  • @borderWidth:1px
.ghostMixin(@color1,@color2,@height,@fontSize,@radius,@fontFamily,@borderWidth)

Example:

.ghost-btn{
  .ghostMixin();
}
